## ✨ 核心功能

### 🏗️ 完整的 AI 游戏开发团队

- **4 人产品开发团队** - 游戏设计师、全栈工程师、叙事设计师、测试工程师
- **5 人质量保障团队** - 代码审查、安全审计、测试生成、Bug 诊断、进度协调
- **强制 QA 流程** - 未经质量审查的代码不得交付

### 📋 5 阶段开发工作流

1. **创意与设计** - 头脑风暴、系统映射、GDD 编写
2. **架构与规划** - 技术架构、Epic 创建、Sprint 计划
3. **开发与实现** - 功能开发、强制 QA、迭代交付
4. **测试与 QA** - 冒烟测试、回归测试、性能分析
5. **发布与上线** - 发布检查、上线检查、变更日志

### 🎮 多引擎支持

- ✅ Godot 4
- ✅ Unity 2022+
- ✅ Unreal Engine 5

## 🚀 快速开始

### 安装

```bash
# 克隆项目
git clone https://github.com/YOUR_USERNAME/hermes-game-dev-studio.git
cd hermes-game-dev-studio

# 运行安装脚本
./scripts/install.sh
```

### 使用

```bash
# 开始头脑风暴
hermes run "帮我头脑风暴一个 Roguelike 卡牌游戏创意"

# 生成 GDD
hermes run "根据刚才的创意生成完整的 GDD 文档"

# 开始开发
hermes run "开始实现卡牌抽取系统"
```
